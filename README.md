# TVZA BrowserLab

Browser-first copy for the future Windows app.

- Entry point: `browser.html`
- Settings: `browser-settings.html`
- TVZA app integration: `https://ti30x.github.io/tvza-app/index.html`
- Public TVZA site: `https://ti30x.github.io/tvza-app/public.html`
- Login: `https://ti30x.github.io/tvza-app/login.html`
- Default search engine: Google
- Default assistant: Claude

The old local TVZA website screens were removed from this copy. BrowserLab now stays focused on the browser shell and connects to the live TVZA app instead of duplicating it locally.

For the later Windows app, add the final app domain/redirect URL in Firebase Authentication as an authorized domain.
